# Interactive Quiz App

A trivia quiz application with timed questions, score tracking, and answer review, built with pure HTML, CSS, and JavaScript.

## Features

- **12 trivia questions** across science, history, technology, and geography
- **Multiple choice** with 4 options per question
- **Progress bar** showing current question out of total
- **15-second timer** per question with color-coded urgency
- **Score tracking** with live updates
- **Results screen** with percentage ring animation and emoji feedback
- **Answer review** showing correct/incorrect answers with explanations
- **High score** saved to localStorage
- **Smooth transitions** between screens and questions
- **Dark theme** with green/red feedback colors

## Usage

Open `index.html` in any modern browser. Click "Start Quiz" to begin. Answer each question within 15 seconds. Review your answers at the end.

## Tech Stack

- HTML5
- CSS3 (custom properties, SVG animation, keyframes)
- Vanilla JavaScript (localStorage, DOM manipulation)
